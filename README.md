# Updated Website

This folder contains the updated Endo website as a single HTML file.

- `index.html` is the deployable website.
- The page behaves like a multi-page site using hash routes such as `#products`, `#about`, and `#contact`.
- Assets are included in `assets/` for reference and backup. The current `index.html` also inlines the needed CSS/images so it can be opened directly.

To deploy, upload `index.html` as the site homepage. If the host requires it, keep the `assets/` folder beside it.
